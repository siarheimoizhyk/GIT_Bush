TXT
 1. Create an external repository named TXT.
 2. Clone the TXT repository to your local computer.
git clone link
 3. Create a "new.txt" file inside the local TXT.
vim new.txt
 4. Add the file under the git.
git add new.txt
 5. Commit the file.
git commit -m "text"
 6. Push the file to an external GitHub repository.
git push
 7. Edit the content of the file "new.txt" - write information about yourself (full name, age, number of pets, future salary desired). Write everything in TXT format.
vim new.txt
 8. Send changes to external repository.
git add new.txt
git commit -m "text"
git push
 9. Create a preferences.txt file
vim preferences.txt
 10. In the preferences.txt file add information about your preferences (favorite movie, favorite show, favorite food, favorite time of year, party you would like to visit) in TXT format.
 11. Create a file sklls.txt add information about skills that will be studied on the course in TXT format
 12. Make a one-line commit.
git commit -a -m "text"
 13. Send two files to an external repository at once.
git push
 14. On the web interface, create a bug_report.txt file.
 15. Make Commit changes on the web interface.
 16. Modify the bug_report.txt file on the web interface and add the bug report in TXT format.
 17. Make Commit changes (save) the changes in the web interface.
 18. Synchronize external and local TXT repository
git fetch
git pull
